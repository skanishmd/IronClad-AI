# 🤝 Contributing to IronClad AI

First off, thank you for considering contributing to IronClad AI! It’s people like you that make this a great tool for the research community.

## 🛡️ Security First
Since this is a security-focused repository, please follow these rules:
- **NEVER** include your own `GEMINI.md`, `SECRET_IP.md`, or personal credentials in a Pull Request.
- If you are reporting a security vulnerability, please open a private issue or contact the maintainer directly.

## 🚀 How Can I Contribute?

### 🐛 Reporting Bugs
- Use the GitHub "Issues" tab.
- Describe the bug, your OS (Windows/Mac/Linux), and the steps to reproduce it.

### 💡 Suggesting Enhancements
- Open an Issue with the tag `enhancement`.
- Describe why this change would be useful for researchers or developers.

### 🔨 Pull Requests
1. Fork the repo and create your branch from `master`.
2. If you've added logic, please ensure it's documented in the README.
3. Ensure your code follows the existing style (clean, commented PowerShell/Bash).
4. Open the PR and wait for a review!

## 📜 Code of Conduct
By contributing, you agree to maintain a professional and respectful environment for all researchers and developers.
